# CSA B149.1-25 Code Clauses

Complete CSA B149.1-25 Natural Gas Installation Code clauses organized by section and topic.

## Overview

This repository contains the complete set of CSA B149.1-25 code clauses used by the LARK Labs dev-tools MCP server for generating compliant answer keys, code references, and compliance documentation.

## Clause Reference Format

Clauses are referenced using the standard CSA format:

```
Section.Subsection.Clause
Example: 6.5.2 (Section 6, Venting Systems, clause 5, sub-clause 2)
```

Annexes use letter designation:

```
Annex.Section.Clause
Example: C.3.1 (Annex C, Section 3, clause 1)
```
